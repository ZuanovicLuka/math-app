# Math++

Web application for interactive mathematics learning for primary and secondary school students.

## Overview
Math++ provides exercises across curriculum units, daily AI-generated quizzes, progress tracking, and a leaderboard to motivate students through gamification and competition.

## Key Features
- Exercises organized by curriculum units
- Daily quizzes generated using AI
- Progress tracking
- Ranking / leaderboard
- Gamification elements to increase motivation

## Tech Stack
- Frontend: React, Next.js
- Backend: Spring Boot
- Database: Supabase
